# Nucleotide Counter

## Objective

Count the occurrence of each nucleotide in a DNA sequence.

---

## Biological Background

Nucleotide composition provides insight into genomic organization and sequence characteristics.

---

## Example

Input

```
ATGCGCAT
```

Output

```
A:2
T:2
G:2
C:2
```

- Dictionaries
- String methods

## Applications

- Sequence composition analysis
- Comparative genomics
- Quality control
