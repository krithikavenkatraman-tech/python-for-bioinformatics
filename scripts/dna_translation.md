# DNA Translation

## Objective

Translate a DNA coding sequence into an amino acid sequence.

---

## Biological Background

Three nucleotides form one codon.

Each codon specifies one amino acid according to the genetic code.

---

## Example

DNA

```
ATGGCC
```

Protein

```
MA
```
