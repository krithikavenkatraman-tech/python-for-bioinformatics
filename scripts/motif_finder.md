# Motif Finder

## Objective

Locate every occurrence of a motif within a DNA sequence.

---

## Biological Background

Sequence motifs are conserved nucleotide patterns associated with biological function, including transcription factor binding sites and regulatory regions.

---

## Example

DNA

```
ATGCGATGCG
```

Motif

```
ATG
```

Output

```
[1, 6]
```
