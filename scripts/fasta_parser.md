# FASTA Parser

## Objective

Read biological sequences stored in FASTA format.

---

## Biological Background

FASTA is the standard sequence format used in bioinformatics databases such as:

- NCBI
- UniProt
- Ensembl

---

## Example FASTA

```
>Sequence_1

ATGCGTAGCTAG
```

---

## Output

```
Header

>Sequence_1

Sequence

ATGCGTAGCTAG
```
