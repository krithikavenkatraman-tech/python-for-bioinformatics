# Sequence Length Calculator

## Objective

Calculate the total length of a biological sequence.

---

## Biological Background

Sequence length is a basic metric used in genomics and molecular biology to evaluate DNA, RNA, or protein sequences.

---

## Example

Input

```
ATGCGTAGCTA
```

Output

```
11
```
