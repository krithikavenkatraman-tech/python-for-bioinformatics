# GC Content Calculator

## Objective

Calculate the percentage of guanine (G) and cytosine (C) nucleotides in a DNA sequence.

---

## Biological Background

GC content is an important genomic feature influencing:

- DNA stability
- Melting temperature
- Gene regulation
- Evolutionary analysis

---

## Formula

GC % = ((G + C) / Total Bases) × 100

---

## Example

Input

```
ATGCGCGT
```

Output

```
GC Content = 62.50%
```
