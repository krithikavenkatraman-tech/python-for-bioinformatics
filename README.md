# 🧬 Python for Bioinformatics

A curated collection of Python scripts and mini-projects developed during my MSc in Computational Molecular Sciences to automate common bioinformatics tasks and analyze biological data.

## 📌 Objectives

- Apply Python to solve bioinformatics problems
- Practice biological sequence analysis
- Develop reproducible computational workflows
- Build a portfolio of bioinformatics projects

## 🛠️ Technologies

- Python
- BioPython
- Google Colab
- Jupyter Notebook

## 📂 Repository Structure
